# ATM-Project
This project's aim is digital banking operations. This operations are user(sign up, sign in, add money, draw money and send money) and admin(sign up, sign in and follow up users).

# Technology

* Java 17
* Spring Boot 3.2.1
* Spring Web
* Spring Security
* Spring Data Jpa
* Spring Boot DevTools
* PostqreSQL Driver
* Swagger UI
